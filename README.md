# Tech-job-market-analysis
An analysis to find which tech roles dominates the hiring market and where should aspiring professionals focus their upskilling efforts.
This project analyzes market trends in the technology industry using python

# Tools Used
Python
Pandas
Matplotlib

# Key insights
AI Engineer have the highest demand among tech jobs
Machine Learning and Data related roles are highly sought after
The tech industry is shifting towards AI and Data Driven Solutions

# Visualizations
A pie chart was used to show the distribution of job titles. The image of the pie chart is named highest_tech_jobs_in_demand

# Dataset
AI Job Market Dataset from Kaggle
